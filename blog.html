  • 8/8/2024 - Possible server type change?

    Hello, been awhile since the last blog post, anyway because school will be starting up and people are more likely to actually want to play, I'm going to restart the server world. With that being said I also want to make sure that everyone enjoys the server, which is why I'm holding a poll to have you guys vote on what kind of server you want it to be.
    The current types that you can vote on are...

    A simple, friendly, survival building server.

    • This is what the server started as, where killing, stealing, griefing, etc. are disallowed, there are no faction plugins, and a dynmap is there for everyone to see where anyone is.

    A nation server.
    • This is the type we tried during the summer, where killing, stealing, griefing etc. are only allowed during wars, wars can only start with specific situations, people are encouraged to make a faction with multiple people and stay in one base, people are encouraged to be diplomatic with other nations, and a dynmap is also there. (Just want to say that, I know that this didn't work all too well during this summer, but I imagine during the school year where more people are likely to be more active, it'll be a lot more fun.)

    Or maybe a faction server.
    • We have not done this type whatsoever, this is chaos, killing, stealing, griefing, etc. are allowed, period. People are encouraged to be in a faction with people, and dominate the server. A dynmap would not be there, in order to keep people safe, and their bases hidden.

    You can vote on this by either texting me, telling me in person, telling someone who has contact with me, or (preferably) joining the discord server linked above and voting in the #announcements channel. If you have any questions about this please ask me. The poll ends on August 13th, at 12 PM.
